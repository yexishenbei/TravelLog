### 管理系统技术选型

| 技术栈    | 选型                                         |
| --------- | -------------------------------------------- |
| 前端框架  | React               |
| 打包工具  | Webpack |
| 状态管理  | redux                                        |
| 数据存储  | mockjs              |
| UI 组件库 | antdesign  |
| 路由管理  | react-router  |

### 1.启动命令

npm run start
