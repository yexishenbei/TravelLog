### 管理系统技术选型

| 技术栈    | 选型                                         |
| --------- | -------------------------------------------- |
| 前端框架  | React               |
| 打包工具  | Webpack |
| 状态管理  | redux                                        |
| 数据存储  | LocalStorage（或mockjs）              |
| UI 组件库 | antdesign  |
| 路由管理  | react-router-cache-route  |

### 报错及解决方式
1. 由于node版本过高引起无法正常启动的问题（本机node版本为22，实际使用中node版本应为16，官方推荐为14），在命令行输入以下内容

```
$env:NODE_OPTIONS="--openssl-legacy-provider"
npm run start:mock
```
